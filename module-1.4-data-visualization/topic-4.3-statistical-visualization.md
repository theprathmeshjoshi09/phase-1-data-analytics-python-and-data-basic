# 3. Statistical Visualization

Statistical visualization helps you **understand data deeply**, not just display it.

👉 These charts reveal:

* distribution
* variability
* relationships
* uncertainty

---

## 📊 Data Analyst Perspective

Used in:

* EDA (exploratory data analysis)
* hypothesis testing
* feature understanding

---

# 3.1 Distribution Plots (KDE, PDF)

Shows how data is distributed

---

## 📊 Real Use Case

```
Marks = [10, 20, 20, 30, 40, 50]
```

---

## 📊 Types

* Histogram → frequency
* KDE → smooth curve

---

## 🔥 Insight

👉 Helps identify:

* skewness
* peaks
* spread

---

# 🔍 Interview Thinking

* Why use KDE?
  👉 smooth distribution understanding

---

# 3.2 Box Plot (Outlier Detection)

Displays:

* median
* quartiles
* outliers

---

## 📊 Real Use Case

```
Data = [10, 12, 15, 18, 20, 100]
```

👉 100 is outlier

---

## 📊 Applications

* detecting anomalies
* data cleaning

---

## 🔥 Insight

👉 Most powerful tool for spotting outliers

---

# 🔍 Interview Thinking

* What does box plot show?
  👉 distribution + outliers

---

# 3.3 Violin Plot

Combination of:

* box plot
* distribution

---

## 📊 Applications

* comparing distributions
* understanding density

---

## 🔥 Insight

👉 More detailed than box plot

---

# 3.4 ECDF Plot

Empirical cumulative distribution

---

## 📊 Real Use Case

```
Data = [10, 20, 30, 40]
```

---

## 📊 Insight

👉 Shows % of values below a point

---

## 📊 Applications

* probability understanding
* percentile analysis

---

# 3.5 Confidence Intervals

Shows uncertainty in estimates

---

## 📊 Real Use Case

Average salary = 50k
CI = (45k – 55k)

---

## 🔥 Insight

👉 True value likely lies in range

---

# 🔍 Interview Thinking

* Why confidence interval important?
  👉 measure uncertainty

---

# 3.6 Error Bars

Shows variability

---

## 📊 Real Use Case

| Product | Avg | Error |
| ------- | --- | ----- |
| A       | 100 | ±10   |
| B       | 150 | ±20   |

---

## 📊 Insight

👉 Indicates reliability

---

# 3.7 Regression Plots

Shows relationship between variables

---

## 📊 Real Use Case

---

## 📊 Insight

👉 Shows:

* trend
* correlation

## 🔥 Insight

👉 Used in ML & predictive analysis

# 3.8 Why It Matters

---

## 📊 Example

```
[10, 20, 30, 40, 500]
```

👉 Mean misleading
👉 Box plot shows outlier
👉 KDE shows skew

---

## 📊 Real Insight

👉 Without statistical visualization:

* you miss patterns
* you make wrong decisions
