# 1. What is Data Visualization

Data Visualization is the process of **representing data graphically**
(charts, graphs, maps) to make information **easy to understand and
analyze**.

Instead of reading raw numbers, we use visuals to quickly identify
insights.

---

## 📊 Data Analyst Perspective

Used for:

* identifying trends
* spotting patterns
* detecting anomalies
* presenting insights

---

### Example:

#### Raw Data:

```
Jan: 200
Feb: 250
Mar: 300
Apr: 450
```

👉 Hard to analyze quickly

#### Visualization:

* Line chart → clearly shows **growth trend**

👉 Instantly understandable

# 1.2 Types of Data

---

## 🔹 1. Categorical Data

Represents **labels or groups**

👉 Examples:

* product category
* gender
* region

👉 Best charts:

* bar chart
* count plot

---

## 🔹 2. Numerical Data

### a) Discrete

Countable values

👉 Example:

* number of students

---

### b) Continuous

Measurable values

👉 Example:

* height
* salary

👉 Best charts:

* histogram
* density plot

---

## 🔹 3. Time-Series Data

Data over time

👉 Example:

* stock prices
* sales trends

👉 Best chart:

* line chart

---

## 📊 Real Insight

👉 Choosing correct chart depends on data type
