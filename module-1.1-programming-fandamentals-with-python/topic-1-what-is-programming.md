# 1. What is Programming 

Programming is the process of writing instructions that a computer can execute to perform specific tasks.

A computer understands only **machine code (0s and 1s)**, so programming languages act as a bridge between humans and machines.

## 1.1 What is a Program 

A **program** is a set of instructions written to perform a task.

### Real-world Example:

* ATM system → withdraw money
* E-commerce → calculate total price

---

### 🔧 Practical Example 

```python
prices = [100, 250, 300]
total_price = sum(prices)

print("Total Revenue:", total_price)
```

---

## 1.2 What is an Algorithm

An algorithm is a **step-by-step solution to a problem**.

---

### 📊 Data Analyst Use Case

Problem: Calculate total sales

```
Step 1: Take sales data
Step 2: Add all values
Step 3: Return result
```

---

### 🔧 Python Implementation

```python
sales = [100, 200, 300]
total = 0

for s in sales:
    total += s

print(total)
```

---

## 1.3 Algorithm vs Program 

| Algorithm | Program        |
| --------- | -------------- |
| Logic     | Implementation |
| Planning  | Execution      |

---

## 🔍 Interview Thinking
